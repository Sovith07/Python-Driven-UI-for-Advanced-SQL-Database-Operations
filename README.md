# 📦 Inventory & Supply Chain Analytics Dashboard

A full-stack **Inventory Management and Supply Chain Analytics Dashboard** built using **Python, MySQL, Streamlit, and Pandas**.  
This project helps businesses monitor inventory levels, manage suppliers, track stock movement, automate reorders, and generate real-time analytics through an interactive web dashboard.

---

## 🚀 Project Overview

Managing inventory manually can lead to stock shortages, overstocking, delayed reorders, and poor decision-making.

This project solves that by providing a **data-driven dashboard** where users can:

✅ Monitor inventory performance  
✅ Track suppliers and products  
✅ Analyze sales and restock activity  
✅ Detect low-stock items automatically  
✅ Place and receive reorders  
✅ View product history in real time

## 📊 Key Features

### 🔹 Dashboard Analytics
- Total Suppliers
- Total Products
- Product Categories
- Sales Value (Last 3 Months)
- Restock Value (Last 3 Months)
- Products Below Reorder Level

### 🔹 Inventory Operations
- Add New Product
- Product Inventory History
- Place Reorders
- Mark Orders as Received

### 🔹 Database Features
- Relational schema design
- Joins, Aggregations, Subqueries
- Stored Procedures
- Views
- Transaction workflows
